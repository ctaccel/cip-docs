## 图片平均色调

imageAve接口用于计算一幅图片的平均色调，并以`0xRRGGBB`形式返回。

### imageAve

>**注意：**
>请忽略以下内容中的空格与换行符。

```
imageAve
```
### 参数说明
| Item | Value | Description | Optional |
| ---- | ----- | ----------- | -------- |
| -    | -     | -           | -        |

#### 关于返回值说明

{

​    "RGB": "<ImageAve string>"

}
