## 图片基本信息

图片基本信息包括图片格式、图片大小、色彩模型。

### imageInfo

>**注意：**
>请忽略以下内容中的空格与换行符。

```
imageInfo
```
### 参数说明
| Item | Value | Description | Optional |
| ---- | ----- | ----------- | -------- |
| -    | -     | -           | -        |

#### 关于返回值说明

| mode       | description                          |
| :--------- | :----------------------------------- |
| size       | 文件大小，单位：Bytes                |
| format     | 图片类型，如`png`、`jpeg`、`bmp`等。 |
| width      | 图片宽度，单位：像素(px)。           |
| height     | 图片高度，单位：像素(px)。           |
| colorModel | 彩色空间，如`rgb`、`ycbcr`等。       |
